# Music-Player
Simple HTML, CSS, javascript Example<br><br>
<h1>How to Run</h1><br>
1. Create two folders with name "music" and "images"<br>
2. Rename all music and images with "1", "2", "3", ...<br>
3. Music should be in .mp3 format and images in .jpg<br>
4. You can add more music in music folder and add path in "song_list.js" file<br>
5. Run on local server
